# W3.1: Closure Semantics Fix

## Change

WHEEL and SATELLITE closure now checks ALL 5 position counters:

```python
all_flat = (
    current.shares_held == 0
    and current.open_short_puts == 0
    and current.open_short_calls == 0
    and current.open_long_puts == 0
    and current.open_long_calls == 0
)
```

**Before:** WHEEL checked only `shares_held + short_puts + short_calls`. SATELLITE checked only `long_puts + long_calls`. Both could close with residual positions in unchecked counters.

**After:** Both cycle types use the same `all_flat` check. A cycle stays ACTIVE until ALL positions are zero.
