# W3.2: Non-Negativity Guards

Generated: 2026-04-07

## Change

Every counter decrement in `_apply_event()` now wrapped with `_guard_decrement()`:

```python
def _guard_decrement(current, delta, counter_name, ev):
    result = current - delta
    if result < 0:
        _walker_warnings.append(f"Non-negative guard: {counter_name} would go to {result}...")
        return 0  # clamp to 0
    return result
```
